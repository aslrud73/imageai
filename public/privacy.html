개인정보처리방침

시행일: 2026년 7월 10일 · 마음결 (관계 조율 노트)

한 줄 요약: 마음결은 서버가 없는 앱입니다. 회원가입이 없고, 운영자는 이용자의 어떤 기록도 수집하거나 볼 수 없습니다. 모든 데이터는 이용자의 기기 안에, 그리고 이용자가 원할 경우 이용자 본인의 Google Drive에만 저장됩니다.

1. 수집하는 개인정보

마음결은 개인정보를 수집하지 않습니다.

  • 회원가입, 로그인, 이름·연락처 입력이 없습니다.
  • 작성한 모든 기록(감정 기록, 회고, 약속, 체크인)은 이용자 기기의 브라우저 저장소(localStorage)에만 저장됩니다.
  • 운영자의 서버로 전송되는 데이터가 없으며, 운영자는 이용자의 기록에 접근할 수 없습니다.
  • 광고·분석 도구(쿠키, 트래커)를 사용하지 않습니다.

2. Google Drive 백업 (선택 기능)

이용자가 직접 "Google Drive 연결하기"를 선택한 경우에만 작동하는 기능입니다.

  • Google의 공식 인증(OAuth)을 사용하며, 앱은 이용자의 Google 비밀번호를 알 수 없습니다.
  • 요청 권한은 drive.file(이 앱이 만든 파일에만 접근)로 제한됩니다. 이용자의 다른 드라이브 파일은 읽을 수 없습니다.
  • 백업 파일은 이용자 본인의 Google Drive에만 저장되며, 운영자는 접근할 수 없습니다.
  • 연결된 Google 계정 이메일은 연결 상태 표시 용도로 이용자 기기 안에만 저장됩니다.
  • 연결은 설정에서 언제든 해제할 수 있고, 드라이브의 백업 파일은 이용자가 직접 삭제할 수 있습니다.
  • 본 앱의 Google 사용자 데이터 활용은 Google API 서비스 사용자 데이터 정책(제한적 사용 요건 포함)을 준수합니다.

3. 공유 기능

  • 기록 공유는 이용자가 항목을 직접 선택했을 때만 이루어집니다.
  • 공유 코드는 이용자가 정한 교환 PIN으로 기기 안에서 암호화되어 생성되며, 어떤 서버에도 전송·저장되지 않습니다. PIN 없이는 코드를 열 수 없습니다.
  • 코드의 전달(메신저 등)은 이용자가 선택한 경로를 통합니다.

4. 데이터의 보관과 삭제

  • 기록은 이용자가 삭제하기 전까지 이용자 기기에 보관됩니다.
  • 설정의 "모든 데이터 삭제"로 즉시, 완전히 삭제할 수 있습니다 (운영자에게 남는 사본 없음).
  • 브라우저 데이터 삭제 시 기록이 사라질 수 있으므로 백업 기능 이용을 권장합니다.

5. 제3자 제공

운영자는 이용자의 데이터를 보유하지 않으므로, 제3자에게 제공하거나 판매하는 일 자체가 불가능합니다.

6. 아동의 개인정보

마음결은 성인(부부·연인)을 위한 서비스이며, 아동을 대상으로 하지 않고 아동의 개인정보를 수집하지 않습니다.

7. 이용 시 유의사항

마음결은 부부상담, 정신건강 진단, 법률 판단, 긴급 구조 서비스를 대체하지 않습니다. 폭력·협박 등 안전이 우려되는 상황에서는 기록보다 안전 확보와 전문기관의 도움이 우선입니다.

8. 국외 서비스 이용 안내

운영자는 이용자의 개인정보를 수집·이전하지 않습니다. 다만 이용자가 선택적으로 Google Drive 백업을 연결하는 경우, 백업 파일은 이용자 본인의 Google 계정(Google LLC, 미국)에 저장됩니다. 이는 이용자가 자신의 계정에 직접 저장하는 것으로, 연결·해제와 파일 삭제 모두 이용자가 직접 관리합니다.
