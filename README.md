# 우리반 이미지 메이커 (Classroom Image Maker)

교사용 수업 이미지 생성 웹앱. 유형과 사이즈를 고르면 무료 AI 이미지 API로 수업 자료용 이미지를 만들어 줍니다.

## 특징

- **자유 설명 입력**: 만들고 싶은 이미지를 그대로 설명하면 스타일 강제 없이 충실하게 생성 (스타일을 원하면 설명에 함께 적기 — "동화책 그림체로" 등)
- **사이즈 선택**: 정사각형 · 가로(4:3, 16:9) · 세로(3:4) · A4, 표준/고화질(인쇄용) 전환
- **미리보기 → 저장** 흐름, "다른 그림"으로 변형 탐색
- **수정**: "배경을 밤하늘로" 같은 보정 문구를 입력하면 같은 구도(seed 유지)에서 이미지를 다듬어 재생성
- **업스케일**: 클릭 한 번으로 같은 이미지를 2배 해상도로 재요청 (인쇄용)
- **반응형**: PC · 태블릿 · 모바일 브라우저에서 모두 동작
- **교체 가능한 백엔드**: 무료 API(Pollinations, 키 불필요)로 시작하되, `src/api/imageProvider.js` 한 곳만 바꾸면 안전 필터가 강한 유료 제공자로 교체 가능

## 안전 설계 (교사용)

- 부적절 키워드 입력 차단(최소 방어선)
- 학생 개인정보 입력 방지 안내
- 생성물은 **교사가 확인 후 사용**하는 것을 전제 (아이 직접 노출 없음)
- 생성 이미지는 **저장해서 재사용** 권장 (무료 API 중단 대비)

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드 (dist/)
npm run preview  # 빌드 결과 미리보기
```

> 참고: 이미지 생성 요청은 사용자(교사)의 브라우저에서 직접 API로 전송됩니다.

## 구조

```
src/
  App.jsx                 메인 UI (유형/주제/사이즈 → 생성 → 결과)
  components/ImageCard.jsx 개별 이미지 카드 (로딩·수정·업스케일·다운로드)
  components/Icons.jsx    인라인 SVG 아이콘 세트
  data/presets.js         유형·사이즈 프리셋 + 안전 필터
  api/imageProvider.js    이미지 백엔드 어댑터 (교체 지점)
```

## 향후 확장

- 인쇄 레이아웃 자동 배치 (A4에 그림카드 격자)
- 이미지 아래 단어(한글/영어) 자동 삽입
- 테마 세트 일괄 생성 (예: 동물 10종)
- 내 자료실 (저장·재사용), PDF 내보내기
- 한글 주제 → 영어 프롬프트 자동 번역 단계
